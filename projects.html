Group Projects

Top Shopper

This game was created as the term long project for CPSC 585 (Games Programming) at the University of Calgary. Our group consisted of five students. We had to develop a game fitting the racing genre. The game was built for Windows in Visual Studio 2017 using C++, GLSL, OpenGL, GLFW, glad, GLM, stb, FreeType, PhysX, and SDL2.
