University

Top Shopper

This game was created as the term long project for CPSC 585 (Games Programming) at the University of Calgary. Our group consisted of five students. We had to develop a game fitting the racing genre. The game was built for Windows in Visual Studio 2017 using C++, GLSL, OpenGL, GLFW, glad, GLM, stb, FreeType, PhysX, and SDL2.

Mouse Racer

My first Blender animation created for CPSC 587 (Fundamentals of Computer Animation) at the University of Calgary.

Arkanoid

Arkanoid game clone written in ARMv7 Assembly to run on a Raspberry Pi 3 Model B with SNES controller input. It was created as an individual assignment for CPSC 359 (Computing Machinery II) at the University of Calgary.
